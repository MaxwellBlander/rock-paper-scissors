Rock Paper Scissors!

This game is run in the dev console by running the function playGame(). To play the game right click anywhere on the screen and select inspect. Then navigate to the tab where it says console or press the esc key to open the console drawer with the elements tab open. In the text box for the console type playGame() then press enter and select your choice. The game will end and declare a winner after 5 rounds. To play again retype playGame() and press enter in the console.
